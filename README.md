## Run Locally

Clone the project



Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the client(Frontend)

```bash
  npm  start
```

Start the server(Backendend)

```bash
  npm  run dev
```
Once go through package.json for all the dependencies
